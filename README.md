# NotionPlay

轻量级游戏简谱编辑器、游戏乐器自动化工具

# 特点

- 支持简谱编辑，其中对于乐理相关内容进行了简化，即便你不太懂乐理也能轻松上手

- 支持按键模拟，可以将简谱转换为按键操作序列以在游戏中自动演奏

## Ⅱ. 快照

> 转换可视简谱为按键操作序列的过程称为快照，这可以减少游戏中自动演奏的性能占用

(1) 工具栏自左向右第二个选项可以切换当前模式为 `编辑模式` or `模拟模式` ，游戏内播放需要切换至模拟模式 ，并使用你在编辑模式下创建的`快照`

(2) 使用 `Ctrl + Shift + V` 打开/关闭游戏内弹窗，于是可以开始自动演奏

![](https://s3.bmp.ovh/imgs/2025/05/20/c8d246f039201a15.png)
